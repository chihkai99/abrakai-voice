📱 手機 App 化使用說明

iPhone 可把本網站加入主畫面，之後像 App 一樣從桌面圖示開啟。建議使用 Safari 或 Chrome 的分享按鈕，選「加入主畫面」。

⚙️ 智慧路由與 API 金鑰戰情中心

正式解析規則：錄音轉文字、PDF/Word 提純、AI 報告產生，必須先連接 Cloudflare Worker 後端。API Key 不應放在公開前端，也不應提交到 GitHub。

🛡️ 正式 AI 後端

這個網址只需要設定一次。設定成功後，前端會把錄音檔或文件送到 Worker，由 Worker 使用 Secrets 裡的 API Key 呼叫 AI 供應商。

🔑 本機 API 節點紀錄

此區可記錄您手上的 API Key 與模型配置；正式呼叫仍以 Worker Secrets 為主。若要測試節點，系統會透過 Worker 測試，不會讓前端直接打供應商 API。
